## 💡 Why this exists

**Background:** Engineered to facilitate a comprehensive and seamless user experience for the discovery of healthcare professionals. The underlying architecture provides sophisticated real-time data filtering, predictive autocomplete search functionality, and deterministic URL state synchronization to maintain persistent session fidelity.

## ✨ Features

- **Real-time Search & Autocomplete:** Search doctors by name, clinic, or specialty with instant autocomplete suggestions.
- **Advanced Filtering & Sorting:** Filter doctors by specialty and consultation mode (Video/In Clinic). Sort results by experience or consultation fees.
- **URL State Synchronization:** Automatically syncs search and filter parameters to the URL for easy sharing and persistence.
- **Dynamic Avatar Generation:** Generates beautiful, color-coded initials avatars for doctors lacking profile pictures.

## 🚀 Quick Start

> [!IMPORTANT]
> Ensure you have Node.js and your preferred package manager installed before proceeding.

### Prerequisites

| Requirement | Version | Notes |
|-------------|---------|-------|
| Node.js | >= 18.0.0 | JavaScript runtime environment |
| npm | >= 8.0.0 | Node package manager |

### Clone & Setup

```bash
git clone https://github.com/D4rk-Pho3nix/my-app.git
cd my-app
```

**Install**
```bash
npm install
```

**Build**
```bash
npm run build
```

**Run**
```bash
npm run dev
```

## 📖 Usage

Run the development server and navigate to the local URL (typically `http://localhost:3000`) to search and filter through the available list of doctors.

```bash
# Start the development server
npm run dev
```
